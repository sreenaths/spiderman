# SpiderMan
Spider dataset made easy - A user's manual

## Install Requirements
Python 3.11.4 was used for development.

```
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## Build Dataset

[BUILD_README.md](BUILD_README.md) must help you to build SpiderMan from the original Spider data.

## View Dataset Stats
```
python scripts/print_stats.py
```
