# SpierMan - Scripts

## Build Dataset
Spider data is alredy available in plain text format under `/dataset` directory. But in case you would like to rebuild following must help.

Download spider.zip from [here](https://drive.google.com/uc?export=download&id=1TqleXec_OykOYFREKKtschzY29dUcVAQ), into the `./source` directory.
```
python build_dataset.py
```

## Load Data
```
python load_data.py
```
